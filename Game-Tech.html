GAME TECH

"Build, Play, Innovate."

Menciptakan pengalaman digital yang interaktif, edukatif, dan berdampak.

Tentang Divisi

Divisi GameTech MBC Lab adalah tempat di mana kreativitas bertemu teknologi. Kami mengeksplorasi dan mengembangkan berbagai jenis game—mulai dari game edukasi, simulasi, hingga hiburan dengan pendekatan interdisipliner antara desain, pemrograman, dan psikologi pemain.

Di sini, kamu bisa belajar membuat game dari nol, mulai dari prototyping, desain antarmuka, hingga deployment lintas platform.

Fokus

  1. Game Development, pembuatan game 2D & 3D menggunakan Unity, Unreal Engine, atau Godot.
  2. Game Design, konsep gameplay, balancing, narrative design, dan gamification.
  3. Asset Creation, ilustrasi karakter, animasi sprite, modeling 3D, dan sound FX.
  4. Cross-Platform Deployment, publish ke Android, WebGL, PC, dan konsol.
  5. Game Analytics, pengumpulan data pemain untuk meningkatkan pengalaman dan retensi.

Prestasi

  1. Proyek game edukasi “Smart Mosque Adventure” berbasis Unity.
  2. Riset tentang Gamifikasi Pembelajaran dan aplikasi berbasis storytelling interaktif.
  3. Workshop: Game Dev Bootcamp, Pixel Art Class, dan Unity Day.
  4. Showcase & Testing Day di berbagai event kampus dan publik.
